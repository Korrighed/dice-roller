# Dice Roller

3D dice roller for D&D and TTRPG — built with React 19, Vite, and Three.js.

## Features

- Polyhedral dice: D4, D6, D8, D10, D12, D20, D100
- Real-time 3D animation with face-up detection
- Roll modifier support (+/– N)
- Persistent roll history (localStorage)

## Stack

- React 19 + TypeScript
- Vite
- Three.js + React Three Fiber

## Getting Started

```bash
npm install
npm run dev       # http://localhost:5173
npm run build     # production build → dist/
npm run preview   # serve the dist/ build locally
```

## Deployment

Hosted on Netlify. Deploys automatically on push to `main`.
